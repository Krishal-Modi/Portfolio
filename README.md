## 📖 How to Use

### View Live
Visit the live portfolio: [https://github.com/Krishal-Modi/Portfolio](https://github.com/Krishal-Modi/Portfolio)

### Run Locally
1. Clone the repository:
```bash
git clone https://github.com/Krishal-Modi/Portfolio.git
cd Portfolio
```

2. Open in browser:
```bash
# Using Python (simple HTTP server)
python -m http.server 8000

# Then visit: http://localhost:8000
```

3. Or simply open `index.html` in your browser
